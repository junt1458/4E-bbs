# 岐阜高専 電気情報工学科 4年 情報工学実験 課題3 応用課題
情報工学実験の応用課題のソースコードです。

## コマンド
```bash
# 起動
$ docker-compose up -d --build

# 終了
$ docker-compose down
```

## 初回起動設定
1. docker-compose.sample.ymlをdocker-compose.ymlにコピー
1. docker-compose.ymlの5行目`***CHANGE_HERE***`を任意のパスワードに変更
1. 起動コマンドを実行

## 言語/フレームワーク
- Next.js
- PHP7
